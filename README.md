# Life
Life는 쉽게 게시판 형 사이트를 구축할 수 있는 [UPPERCASE](http://UPPERCASE.IO)와 [NSP](http://nsp.js.org) 기반의 CMS(Content Management System)입니다.

## 로고 및 아이콘
아래 세가지 파일을 사이트 컨셉에 맞게 만들어, `Life/R`폴더에 넣어줍니다.
* `logo.png`
* `favicon.ico`
* `apple-touch-icon.png` (크기를 180x180으로 만듭니다.)
